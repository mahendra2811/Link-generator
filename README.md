# Link Generator

## Overview
Link Generator is a web application that allows users to create shareable links from phone numbers and text messages, EX. WhatsApp.

## Technologies Used
- HTML
- CSS
- JavaScript
- React

## Usage
1. Clone the repository.
2. Open the `index.html` file in your web browser.
